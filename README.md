>

- ✨ Features :
   - Basic arithmetic operations (+, –, ×, ÷)
   - Advanced operations:
      - Power (xⁿ)
      - Square root (√x)
      - Percentage (%)
      - Factorial (x!)
   - Clear and Delete functionality
   - Keyboard input support
   - Responsive layout
   - Clean and modern UI
